
# Figma Make란?

- **Figma Make**는 AI 프롬프트를 활용해 가상의 앱을 기획할 수 있는 Figma의 기능입니다.
    
- 사용자가 입력한 프롬프트를 분석하여 **UI와 동작을 코드 기반의 앱 형태로 자동 생성**해 줍니다.
    

---

## Figma Make로 앱 기획 시 유의사항

- 내용 수정 과정에서 다음과 같은 문제가 발생할 수 있습니다.
    
    - 기존에 정상 동작하던 코드가 꼬이거나 오류가 발생함
        
    - 바로 이전 프롬프트까지 반영되었던 기능이 갑자기 사라져 동작하지 않는 경우
        
- 따라서 **단계별·점진적 기획**이 안정적입니다.

## Figma Make 인터페이스

### 1. AI 프롬프트 영역

- 사용자가 원하는 앱의 기능과 화면 구성을 텍스트로 입력하는 영역입니다.
    
- 입력된 프롬프트를 분석하여:
    
    - 화면 및 기능을 자동으로 생성하고
        
    - 적용된 내용을 프롬프트 상단에 요약 설명해 줍니다.
        

### 2. 디버깅 툴

- 다음 기능으로 구성되어 있습니다.
    
    - 미리보기 새로고침
        
    - 콘솔 창
        
    - 모바일 형태 미리보기
        

### 3. 기기 선택 & 화면 크기 조절

- **좌측**: 기기 종류 선택  
    → 선택한 기기의 화면 크기에 맞춰 미리보기 제공
    
- **중앙**: 너비와 높이를 직접 설정  
    → 특정 기기에 국한되지 않고 자유로운 화면 크기 조절 가능
    

### 4. 내용 화면 (미리보기 / 코드)

- 5번 탭에서 선택한 항목에 따라:
    
    - 미리보기 화면 또는
        
    - 코드 화면이 표시됩니다.
        

![[Pasted image 20251217160438.png]]  
![[Pasted image 20251217160456.png]]

### 5. 미리보기 / 코드 전환 탭

- 화면 결과와 생성된 코드를 자유롭게 전환하며 확인할 수 있습니다.

## Figma Make로 기획하는 방법

- **완성된 기획이 없는 상태라면**
    
    - 전체 화면을 한 번에 만들기보다는  
        👉 **한 페이지씩 세부적으로 기획**하는 방식이 더 안정적입니다.
        

> ※ 본 내용은 _Facility Finder_ 기획 사례를 기반으로 작성되었습니다.
